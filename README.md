# Asana (asana)
Asana is a web and mobile application designed to help teams organize, track, and manage their work. The Asana API allows developers to programmatically access and integrate Asana's project management capabilities into their applications. Asana provides 30+ REST API resources covering tasks, projects, portfolios, goals, teams, workspaces, webhooks, and automation rules, enabling deep integration with enterprise workflows.
